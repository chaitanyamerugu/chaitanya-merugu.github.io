03 — Core Components

Technical Architecture

Component 01

ABAP OO Engine

The automation core is built using object-oriented ABAP, encapsulating payroll logic into reusable classes. This approach ensures maintainability, testability, and clean separation of concerns across the processing layers.

ABAP OO · Classes · Methods
Component 02

PNP / PNPCE Logical Database

Leverages SAP's PNP and PNPCE logical databases for optimised HR master data extraction — providing pre-joined access to infotypes, payroll results, and employee data without custom SQL overhead.

PNP · PNPCE · Infotypes
Component 03

Payroll Schema Integration

Integrates with country-specific payroll schemas and functions, executing wage type processing, deductions, tax calculations, and net pay determinations across all 43 supported country configurations.

Schemas · Functions · Wage Types
Component 04

ALV Grid + File Transfer

Generates structured ALV grid reports for in-system payroll review, then automatically packages and transfers final payroll documents to the CRM bank file system — eliminating the manual download-and-upload loop.

ALV · File Transfer · CRM
04 — Implementation Process

Build Sequence

1

ABAP OO Architecture Design

Designed the object-oriented class hierarchy for the payroll engine, defining interfaces for HR data extraction, schema execution, output generation, and file transfer as distinct, independently testable layers.

2

PNP/PNPCE Data Extraction Layer

Implemented the HR data extraction layer using PNP and PNPCE logical databases, configuring selection screens, infotype reads, and payroll result cluster access for each employee population.

3

Payroll Schema & Cluster Integration

Integrated country-specific payroll functions and schemas, mapping wage type tables and cluster data export for gross-to-net calculations across all 43 client country configurations.

4

ALV Grid Report Implementation

Built interactive ALV grid output for payroll result review, including field catalogues, sort options, subtotals, and exception highlighting — enabling pre-upload validation by payroll teams.

5

Automated CRM File Upload

Implemented automated file transfer to the CRM bank file system, replacing the manual download-upload workflow. Payroll documents are packaged, named, and deposited post-run without user intervention.

6

Multi-Country Testing

Tested the full pipeline across multiple country-specific payroll configurations, validating schema execution, tax logic, and output format compliance for each ADP client locale.
